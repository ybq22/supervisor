# 🚀 快速参考

## 基本用法

```bash
# 全面收集信息（默认，推荐）
/distill-mentor "导师姓名"

# 指定机构
/distill-mentor "导师姓名" --affiliation "机构名称"

# 快速模式（仅 ArXiv + API）
/distill-mentor "导师姓名" --no-browser
```

## 信息收集维度

浏览器搜索从 **4 个维度**收集信息：

1. **🎓 个人主页和学术信息**
   - 机构页面
   - 实验室网站
   - Google Scholar

2. **📄 论文和出版物**
   - ArXiv 论文
   - DBLP 出版物
   - ResearchGate

3. **🎤 演讲和访谈**
   - TED talks
   - YouTube 视频
   - 会议演讲

4. **📚 Wikipedia 和百科**
   - 人物传记
   - 成就介绍
   - 学术贡献

## 数据质量评分

| 分数 | 等级 | 说明 |
|------|------|------|
| 0.8-1.0 | ⭐⭐⭐⭐⭐ 优秀 | 可生成高质量数字分身 |
| 0.6-0.8 | ⭐⭐⭐⭐ 良好 | 建议补充更多材料 |
| 0.4-0.6 | ⭐⭐⭐ 一般 | 需要提供额外信息 |
| < 0.4 | ⭐⭐ 不足 | 建议手动补充材料 |

## 常见问题

**Q: 为什么默认使用浏览器搜索？**
A: 为了收集更全面的信息，生成更准确的数字分身。

**Q: 如何使用快速模式？**
A: 添加 `--no-browser` 参数。

**Q: 浏览器搜索失败怎么办？**
A: 系统会自动回退到 DuckDuckGo API，确保继续工作。

**Q: 数据质量评分低怎么办？**
A: 可以手动上传导师的简历、论文列表等补充材料。
